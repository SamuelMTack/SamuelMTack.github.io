How to preview locally

Open index.html in your browser, or run a small local static server from the repo root.

npx http-server .
# or
python -m http.server 8000

Then open http://localhost:8080 (or :8000 for Python) in your browser.

Examples of small edits

  • Change the body text in index.html.
  • Add a stylesheet under css/ and reference it from the head: <link rel="stylesheet" href="css/styles.css">.
  • Add images under assets/ and reference them with relative URLs.

Notes for contributors

Prefer minimal, focused PRs. If you propose adding a build system or dependencies (e.g., package.json), include a short README that explains why and how to run the build locally.
